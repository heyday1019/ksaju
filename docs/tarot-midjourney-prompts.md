## 사용 방법

### Midjourney에서 사용하기
1. 원하는 카드 프롬프트를 복사
2. `[MASTER STYLE]` 부분을 상단의 **마스터 스타일 프롬프트** 전체로 교체
3. `/imagine` 명령어로 생성

### 일관성 유지 팁
- 같은 세션에서 작업하거나 `--seed` 값을 고정해 스타일 일관성 유지
- 카드 생성 후 마음에 들면 `--sref` (style reference)로 그 이미지를 레퍼런스로 활용
- 첫 메이저 아르카나 5장 먼저 생성해 스타일 확인 후 나머지 진행 권장
